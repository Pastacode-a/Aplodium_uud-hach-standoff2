# Aplodium_uud-hach-standoff2
Aplodium - это относительно новый чит в игре Standoff2, его разработчики (flacon, aks1ds) из Lua скриптов перешли в C++. Вот еще недавно писали Lua скрипты для gameguardian, а сейчас уже пишем читы на углекислый газ 2.
Основные функции: ESP, AimBot, visuals(FOV Editor, sky color), misc(watermark), exploits(AirJump)
